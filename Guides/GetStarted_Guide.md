# Get Started Guide

When working with NeTEx profiles, you will encounter several common frames:

- [ServiceFrame](../Frames/ServiceFrame/Description_ServiceFrame.md): Contains routes, lines, and scheduled stop points.
- [FareFrame](../Frames/FareFrame/Description_FareFrame.md): Defines fare structures and pricing information.
- [ResourceFrame](../Frames/ResourceFrame/Description_ResourceFrame.md): Holds shared resources like operators and organizations.
- [TimetableFrame](../Frames/TimetableFrame/Description_TimetableFrame.md): Provides detailed timetables for services.
- [SiteFrame](../Frames/SiteFrame/Description_SiteFrame.md): Describes physical sites and stop places.
- [VehicleScheduleFrame](../Frames/VehicleScheduleFrame/Description_VehicleScheduleFrame.md): Contains vehicle schedules and duties.
- [ServiceCalendarFrame](../Frames/ServiceCalendarFrame/Description_ServiceCalendarFrame.md): Defines operational calendars and day types.

Follow these links to learn more about each frame. Other instructions remain unchanged.
