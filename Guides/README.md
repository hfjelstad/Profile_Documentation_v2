# Guides

## Planlagte guider

- **ServiceJourneyPattern og RouteLink-struktur**: Beskriver hvordan ruter og lenker mellom stoppesteder defineres i NeTEx.
- **AccessRightAssignment og Tilgangsrettigheter**: Forklarer hvordan tilgangsrettigheter og restriksjoner håndteres.
- **FareStructure og Tariffsoner**: Gir innsikt i hvordan takstsoner og prisstrukturer modelleres.
- **DeadRun og VehicleService**: Dekker ikke-passasjerkjøringer og kjøretøytjenester.
- **OperationalContext og DayType**: Viser hvordan driftskontekst og dagtyper defineres.
- **Interchange med tidsavhengighet**: Beskriver hvordan korrespondanser med tidsavhengighet settes opp.
- **StopPlace og Quay-detaljer**: Detaljer om stoppesteder og plattformer.
- **Alternative Services (OnDemandTransport)**: Hvordan fleksible transporttjenester modelleres i NeTEx.
