# InterchangeOnly Guides

This folder contains guides for modeling **mandatory interchanges** in NeTEx using `StopUse = interchangeOnly` and `Interchange`.

## Purpose
Use these guides when you need to:
- Restrict a stop to transfer-only usage.
- Model planned or guaranteed interchanges between journeys.
