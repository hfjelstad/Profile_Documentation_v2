## Conventions (to keep this index maintainable)

- Location: place example files under `Objects/<ObjectName>/`.
- Naming: use `Example_<ObjectName>.xml`. For variants, use `Example_<ObjectName>_<Qualifier>.xml` (e.g., `Example_DatedServiceJourney_Extended_02_Replacement.xml`).
- Description: prefer short, object-centric descriptions (e.g., Operator, VehicleType, Route). Add a brief qualifier for variants when helpful.
- When you add a new example, insert a new row in the table above keeping the same format and sorting by Object name.
